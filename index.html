Collection database for GenesysRef. If you would like to contribute to a hosted collection or submit your own, and know how to use GitHub, fork this repo and make a pull request with your changes. If you would like to contribute to a hosted collection or submit your own, and you aren't familiar with GitHub, join the conversation on our Discord. If you would like to use this as an API for Genesys data, please contact SilentArctic through Discord.
